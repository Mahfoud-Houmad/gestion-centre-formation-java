# Gestion d’un Centre de Formation – Java Console

## Description
Application Java en mode console permettant de gérer un centre de formation.
Le projet est basé sur la programmation orientée objet.

## Fonctionnalités
- Gestion des étudiants
- Gestion des formateurs
- Gestion des formations
- Inscriptions des étudiants
- Affichage via menu console

## Technologies
- Java
- Programmation Orientée Objet (POO)
- UML

## Exécution
1. Cloner le projet
2. Ouvrir dans un IDE 
3. Lancer la classe `Main`
